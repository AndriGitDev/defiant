# defiant
"BREACH TIMELINE" - Live CVE & Incident Tracker
Interactive 3D timeline of security vulnerabilities and breaches.
Features:

Real-time CVE data from NVD API
Major breach database (compile historical data)
3D floating timeline (like your Three.js background)
Filter by: severity, vendor, type, date range
Click nodes for full technical details
Severity visualization with colors/animations
Search specific products/vendors
RSS feeds from security blogs

Why it's impressive: Stunning visualization + valuable reference tool. Could become THE go-to CVE explorer.
